## Entregáveis

### `jedi_library/slug.py`

```python
import re
import unicodedata

def normalize(text: str) -> str:
    """NFKD → ASCII → lower → kebab."""
    nfkd = unicodedata.normalize("NFKD", text)
    ascii_text = nfkd.encode("ascii", "ignore").decode("ascii")
    lower = ascii_text.lower()
    return re.sub(r"[^a-z0-9]+", "-", lower).strip("-")

def unique_slug(text: str, existing: set[str]) -> str:
    """normalize(text) + sufixo -N se colisão."""
    base = normalize(text)
    if base not in existing:
        return base
    n = 2
    while f"{base}-{n}" in existing:
        n += 1
    return f"{base}-{n}"
```

### Testes (`test/python/test_slug.py`)

- `normalize("Ação Rápida")` → `"acao-rapida"`
- `normalize("  olá, mundo!  ")` → `"ola-mundo"`
- `normalize("")` → `""`
- `unique_slug("tarefa", set())` → `"tarefa"`
- `unique_slug("tarefa", {"tarefa"})` → `"tarefa-2"`
- `unique_slug("tarefa", {"tarefa", "tarefa-2"})` → `"tarefa-3"`
